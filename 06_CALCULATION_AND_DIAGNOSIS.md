# 計算式・診断ロジック仕様

## 1. 単位

総務省/e-Statの決算統計では金額が千円単位である可能性が高い。実装では単位を必ずメタデータから確認すること。

本仕様では以下を前提にする。

- 金額: 千円
- 水量: m³
- 人口: 人

円/m³を計算する場合、金額が千円なら `×1000` する。

## 2. 主要計算式

### 2.1 使用料単価

```text
使用料単価（円/m³） = 下水道使用料収入（千円） × 1000 ÷ 年間有収水量（m³）
```

### 2.2 汚水処理原価

```text
汚水処理原価（円/m³） = 汚水処理費（千円） × 1000 ÷ 年間有収水量（m³）
```

### 2.3 経費回収率

```text
経費回収率（％） = 下水道使用料収入 ÷ 汚水処理費 × 100
```

または

```text
経費回収率（％） = 使用料単価 ÷ 汚水処理原価 × 100
```

両者が大きく乖離する場合は、単位・水量・打切決算・欠損を疑い、data_quality_statusを`warning`にする。

### 2.4 経費回収率100％達成に必要な改定率

```text
必要改定率100 = 汚水処理費 ÷ 下水道使用料収入 - 1
```

同義：

```text
必要改定率100 = 100 ÷ 経費回収率 - 1
```

例：経費回収率80％なら `100 / 80 - 1 = 0.25`、つまり25％。

### 2.5 経費回収率80％達成に必要な改定率

```text
必要改定率80 = 0.8 × 汚水処理費 ÷ 下水道使用料収入 - 1
```

または

```text
必要改定率80 = 80 ÷ 経費回収率 - 1
```

既に80％以上の場合は0以下になり得るため、画面では「達成済み」と表示する。

### 2.6 使用料単価150円/m³達成に必要な改定率

```text
必要改定率150 = 150 ÷ 使用料単価 - 1
```

既に150円/m³以上の場合は「達成済み」と表示する。

### 2.7 有収率

```text
有収率（％） = 年間有収水量 ÷ 汚水処理水量 × 100
```

### 2.8 水洗化率

```text
水洗化率（％） = 水洗便所設置済人口 ÷ 処理区域内人口 × 100
```

## 3. 使用料適正判定

```typescript
function getFeeAdequacyLabel(expenseRecoveryRate: number | null, feeUnitPrice: number | null): string {
  if (expenseRecoveryRate == null) return '判定不可';
  if (expenseRecoveryRate >= 100) return '適正水準';
  if (expenseRecoveryRate >= 90) return 'やや不足';
  if (expenseRecoveryRate >= 80) return '要注意';
  if (expenseRecoveryRate < 80 && feeUnitPrice != null && feeUnitPrice < 150) return '重点監視';
  return '改定圧力高';
}
```

## 4. 会計収支判定

### 4.1 法適用

```text
経常損益 = 経常収益 - 経常費用
```

- 経常損益 >= 0: 経常黒字
- 経常損益 < 0: 経常赤字
- 純損益 >= 0: 純利益
- 純損益 < 0: 純損失

### 4.2 法非適用

```text
収益的収支比率 = 総収益 ÷ (総費用 + 地方債償還金) × 100
```

- 実質収支 >= 0: 実質収支黒字
- 実質収支 < 0: 実質収支赤字
- 収益的収支比率 >= 100: 収益的収支は均衡以上
- 収益的収支比率 < 100: 収益的収支は不足

## 5. 改定リスクスコア

MVPでは以下のスコアリングを採用する。

| 指標 | 条件 | 点数 |
|---|---:|---:|
| 経費回収率 | 100％未満 | +20 |
| 経費回収率 | 80％未満 | +30 |
| 使用料単価 | 150円/m³未満 | +15 |
| 汚水処理原価 | 直近5年で上昇傾向 | +15 |
| 有収水量 | 直近5年で減少傾向 | +10 |
| 基準外繰入金 | あり | +20 |
| 累積欠損金 | あり | +15 |
| 企業債/地方債償還 | 今後5年で増加見込み | +10 |
| 直近改定 | 10年以上未改定 | +10 |
| 公式改定予定 | あり | スコアではなく別枠表示 |

### 5.1 リスクラベル

| スコア | ラベル |
|---:|---|
| 0〜29 | 改定リスク低 |
| 30〜59 | 中期的に改定可能性あり |
| 60〜79 | 改定リスク高 |
| 80以上 | 早期改定候補・要注視 |

## 6. トレンド判定

5年分以上のデータがある場合のみ計算する。

### 6.1 上昇傾向

単純線形回帰の傾きが正で、かつ初年度比で5％以上増えている場合に上昇傾向とする。

### 6.2 減少傾向

単純線形回帰の傾きが負で、かつ初年度比で5％以上減っている場合に減少傾向とする。

## 7. 異常値フラグ

以下の場合は `flags_json` に保存し、画面に注意表示する。

- 年間有収水量が0またはNULL
- 下水道使用料収入が0またはNULL
- 汚水処理費が0またはNULL
- 経費回収率が300％超
- 使用料単価が500円/m³超
- 汚水処理原価が1000円/m³超
- 前年度比で使用料収入が50％以上増減
- 前年度比で有収水量が50％以上増減
- 法適用移行年度
- 打切決算の可能性
- 事業統合・会計統合の可能性

## 8. 表示コメント生成

### 8.1 適正水準

「最新年度の経費回収率は{rate}%であり、下水道使用料で汚水処理費を概ね回収できている水準です。ただし、将来の更新投資や有収水量の減少により、今後も同水準を維持できるとは限りません。」

### 8.2 やや不足・要注意

「最新年度の経費回収率は{rate}%であり、使用料収入だけでは汚水処理費を完全には回収できていません。経費回収率100％を目指す場合、単純計算では使用料収入を約{required_revision_rate_to_100}%増やす必要があります。」

### 8.3 改定圧力高・重点監視

「最新年度の経費回収率は{rate}%であり、使用料収入不足が大きい状態です。経費回収率100％を目指す場合、単純計算では使用料収入を約{required_revision_rate_to_100}%増やす必要があります。ただし、実際の改定率は審議会、条例改正、一般会計繰入、投資計画等を踏まえて決まります。」
