# 全国下水道使用料適正診断サイト 要件書

## 1. 背景

全国の下水道事業では、人口減少、有収水量の減少、施設老朽化、電気代・委託費・流域下水道維持管理負担金の上昇、企業債償還などにより、下水道使用料の改定検討が必要になる団体が増える可能性がある。

しかし、住民やメディア、自治体職員、研究者が「自分の自治体の下水道使用料水準が妥当なのか」「使用料で汚水処理費を回収できているのか」「どの程度の改定圧力があるのか」を横断的に確認できるサイトは少ない。

本サイトは、総務省/e-Statが公開する地方公営企業決算状況調査を基礎データとして、自治体・事業別の下水道使用料適正度を可視化する。

## 2. サービス名

仮称：全国下水道使用料適正診断

## 3. 対象ユーザー

- 一般住民
- 自治体職員
- 地方議員
- メディア
- 研究者・学生
- インフラ・公営企業に関心のある利用者

## 4. 対象データ

一次データはe-Statの以下。

- 政府統計名: 地方財政状況調査
- 提供統計名: 地方公営企業決算状況調査
- 提供分類1: 全国
- 提供分類2: 調査表
- 提供周期: 年次
- 対象事業: 下水道事業（一）、下水道事業（二）
- 対象区分: 法適用、法非適用

## 5. MVP範囲

MVPでは、以下を必須とする。

- 最新年度1年分の取り込み
- 可能なら過去5年分までの取り込み
- 法適用・法非適用の両対応
- 事業単位のデータ保持
- 自治体単位の画面表示
- 経費回収率、使用料単価、汚水処理原価、必要改定率の算出
- 使用料水準の適正判定
- 会計上の収支判定
- ランキング表示
- 免責表示
- 改定予定情報はCSV/JSONで手動登録できる仕組み

自動クローリングは第2段階とする。

## 6. 表示する主要指標

### 6.1 使用料水準系

- 下水道使用料収入
- 年間有収水量
- 使用料単価
- 汚水処理費
- 汚水処理原価
- 経費回収率
- 経費回収率100％達成に必要な改定率
- 経費回収率80％達成に必要な改定率
- 使用料単価150円/m³達成に必要な改定率

### 6.2 会計収支系

法適用：

- 営業収益
- 営業費用
- 経常収益
- 経常費用
- 経常損益
- 当年度純損益
- 累積欠損金

法非適用：

- 総収益
- 総費用
- 実質収支
- 収益的収支比率

### 6.3 将来リスク系

- 有収水量の過去推移
- 使用料収入の過去推移
- 汚水処理費の過去推移
- 汚水処理原価の過去推移
- 一般会計繰入金
- 基準外繰入金
- 企業債・地方債残高
- 年度別償還予定
- 水洗化率
- 有収率

## 7. 判定区分

### 7.1 使用料適正判定

| 判定 | 条件 | 表示文 |
|---|---:|---|
| 適正水準 | 経費回収率100％以上 | 使用料で汚水処理費を概ね回収できています |
| やや不足 | 90％以上100％未満 | 使用料水準がやや不足しています |
| 要注意 | 80％以上90％未満 | 中期的な使用料改定検討が必要になりやすい水準です |
| 改定圧力高 | 80％未満 | 使用料収入不足が大きく、改定圧力が高い水準です |
| 重点監視 | 80％未満かつ使用料単価150円/m³未満 | 使用料水準・経費回収率の両面で注意が必要です |

### 7.2 会計上の収支判定

法適用：

- 経常損益 >= 0: 経常黒字
- 経常損益 < 0: 経常赤字
- 純損益 >= 0: 純利益
- 純損益 < 0: 純損失

法非適用：

- 実質収支 >= 0: 実質収支黒字
- 実質収支 < 0: 実質収支赤字
- 収益的収支比率 >= 100: 収益的収支は均衡以上
- 収益的収支比率 < 100: 収益的収支は不足

## 8. 画面要件

### 8.1 トップページ

- 自治体名検索
- 都道府県選択
- 「経費回収率が低い自治体」「改定リスクが高い自治体」への導線
- サイトの説明
- 免責文言

### 8.2 自治体一覧ページ

- 自治体名
- 都道府県
- 最新年度
- 経費回収率
- 使用料単価
- 汚水処理原価
- 必要改定率
- 診断ラベル
- 公式改定予定の有無

### 8.3 自治体詳細ページ

- 基本情報
- 事業別カード
- 最新年度の主要指標
- 過去推移グラフ
- 必要改定率
- 会計上の収支判定
- 使用料水準判定
- 一般会計繰入金の状況
- 企業債・地方債の状況
- 公式改定情報
- データ根拠
- 計算式
- 免責文言

### 8.4 ランキングページ

- 経費回収率が低い順
- 必要改定率が高い順
- 使用料単価が低い順
- 汚水処理原価が高い順
- 基準外繰入金依存が高い順
- 都道府県別集計

## 9. 非機能要件

- すべての計算値について根拠データに遡れること
- 年度更新に対応できること
- 調書レイアウト変更に備え、フィールドマッピングを設定ファイル化すること
- 画面表示はスマホ対応
- 計算式と免責文言を明示すること
- 断定的な「値上げ確実」表現は禁止
