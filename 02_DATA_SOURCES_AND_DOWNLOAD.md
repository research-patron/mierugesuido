# 総務省/e-Stat データ取得仕様

## 1. 一次データ

使用する一次データはe-Statで公開されている以下の統計。

- 政府統計コード: `00200251`
- 政府統計名: 地方財政状況調査
- 提供統計名: 地方公営企業決算状況調査
- 提供分類1: 全国
- 提供分類2: 調査表
- 提供周期: 年次
- 統計分野: 行財政 / 財政
- 担当機関: 総務省 自治財政局財務調査課

## 2. e-Stat画面での取得場所

画面から確認する場合：

1. e-Statにアクセス
2. 「地方公営企業決算状況調査」で検索
3. 政府統計「地方財政状況調査」→ 提供統計「地方公営企業決算状況調査」を選択
4. 「全国」→「調査表」→「年次」を選択
5. 対象年度を選ぶ
6. 下水道事業関係の調査表レイアウトおよび各表番号のExcelを取得

## 3. e-Stat API利用

APIを使う場合は、e-StatのアプリケーションIDが必要。

### 3.1 必須環境変数

```bash
ESTAT_APP_ID=xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
```

### 3.2 使用するAPI

- `getStatsList`: 統計表情報取得
- `getMetaInfo`: メタ情報取得
- `getStatsData`: 統計データ取得
- `getDataCatalog`: データカタログ情報取得

ただし、地方公営企業決算状況調査はファイル形式のExcel提供が中心であるため、APIでメタデータを取得し、実ファイルURLを取得できる場合はExcelをダウンロードする。APIで直接数値を取得できない場合は、e-Statのファイル一覧ページまたはデータカタログからExcelリンクを解決してダウンロードする。

### 3.3 discover処理の検索条件

- statsCode: `00200251`
- searchWord: `地方公営企業決算状況調査 下水道事業`
- surveyYears: 対象年度。例: `2025`, `2024`
- collectArea: `1`（全国）
- searchKind: `1`

### 3.4 対象年度

MVPでは以下を対象にする。

- `2025`（令和6年度決算相当の調査）
- `2024`
- `2023`
- `2022`
- `2021`

年度の厳密な対応はファイル名・表題・調査年月を保存すること。

## 5. ファイル保存ルール

Rawファイルは以下に保存する。

```text
data/raw/e-stat/{survey_year}/{accounting_type}/{table_no}_{table_name}_{estat_file_id}.{ext}
```

例：

```text
data/raw/e-stat/2025/legal_applied/20_income_statement_xxxxx.xlsx
data/raw/e-stat/2025/non_legal_applied/26_cash_revenue_expenditure_xxxxx.xlsx
```

## 6. メタデータ保存

ダウンロードしたファイルごとに `source_files` テーブルへ保存する。

- source_file_id
- survey_year
- fiscal_year_label
- government_stat_code
- government_stat_name
- provided_stat_name
- category_1
- category_2
- table_no
- table_name
- accounting_type
- estat_stat_infid
- estat_stats_data_id
- source_url
- local_path
- file_format
- published_at
- downloaded_at
- sha256

## 7. 手動フォールバック

e-Stat APIでファイルURLが取れない場合に備え、以下を許容する。

- `data/raw/manual/{year}/` にExcelを手動配置
- `data/manual/source_files.csv` にファイルメタデータを手入力
- `pnpm etl:import --manual` で取り込む
