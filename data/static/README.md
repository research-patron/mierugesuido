# Static publication data

This directory is generated by `pnpm static:data` from the local development database. It contains only public-source display data needed by the static site. The local database, source download paths, hashes, environment variables, and credentials are intentionally excluded.

Primary provenance remains attached to the displayed values and links to e-Stat, the Ministry of Internal Affairs and Communications, MLIT, and municipality publications. Regenerate and review the complete diff before publication.
